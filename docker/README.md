# Docker Configuration

This directory contains Docker-related configuration files for the BenoCode website project.

## Files

- `Dockerfile.dev` files are located in `backend/` and `frontend/` directories
- `docker-compose.yml` is located in the root directory

## Development Stack

The Docker Compose setup includes:

1. **PostgreSQL 15** - Database service
2. **Backend** - Express.js API server
3. **Frontend** - Next.js application

## Usage

See the main README.md for instructions on how to use Docker Compose for local development.
